JESSX Stock Market Simulator

JESSX is a powerful stock market simulator integrated into EclipseTrader. It allows you to create and run complex trading scenarios, test automated trading strategies using bots, and analyze market behavior in a controlled environment.

Configuration

The JESSX simulation is configured through the default.xml file located in the org.eclipsetrader.jessx/src/org/eclipsetrader/jessx/utils/ directory. This file defines all aspects of the simulation, including:

  • Assets: The securities (stocks) available for trading.
  • Institutions: The market makers or exchanges where assets are traded.
  • Player Types: The different categories of participants in the simulation.
  • Robots (Bots): The automated trading agents that will participate in the simulation. Each bot has a specific trading strategy.
  • News Items: Simulated news events that can influence market behavior.

Running a Simulation

To run a JESSX simulation, you must first have the JESSX broker configured as your default broker. Then, simply click the "Play" button in the main toolbar. This will start the JESSX server and connect the bots and the human player ("ThePlayer") to the simulation.

The simulation will start, and you will see the market data updating in real-time in the watchlists and charts. You can interact with the simulation by placing orders manually, just as you would with a real broker.

Trading Bots

The JESSX simulator includes several types of trading bots, each with a different trading strategy. The bots are defined in the default.xml file and their logic is implemented in Java classes within the org.eclipsetrader.jessx plugin. The primary bots are:

  • Discreet: A bot that trades based on market trends.
  • DiscreetIT: A variation of the Discreet bot.
  • NotDiscreet: A more aggressive trading bot.

The bots will automatically start trading once the simulation begins. You can observe their activity in the orders view.
