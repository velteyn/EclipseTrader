JESSX Stock Market Simulator

JESSX is a powerful stock market simulator integrated into EclipseTrader. It allows you to create and run complex trading scenarios, test automated trading strategies using bots, and analyze market behavior in a controlled environment.

Configuration

JESSX scenarios are standard XML files that define assets, institutions, participant types, robots and optional news/information. You can select and manage scenarios directly in the application:

  • Open Preferences → Plugins → JessX! Scenarios.
  • Pick a bundled scenario (e.g., default.xml, bull-market.xml, bear-market.xml, volatile-market.xml).
  • Or click Browse… to select any .xml file on disk.
  • You can drop custom scenario files into the folder <workspace>/.metadata/.plugins/org.eclipsetrader.jessx/scenarios/; they will appear in the list automatically.

When a bundled scenario is selected, it is copied to the plug‑in state area the first time you run it, so you can further edit it. Scenarios define:

  • Assets: The securities (stocks) available for trading.
  • Institutions: The market makers or exchanges where assets are traded.
  • Player Types: The different categories of participants in the simulation.
  • Robots (Bots): The automated trading agents that will participate in the simulation. Each bot has a specific trading strategy.
  • News Items: Simulated news events that can influence market behavior.

Running a Simulation

Select a scenario in Preferences, ensure the JESSX broker is configured, then click the “Play” button in the main toolbar. This starts the JESSX server and connects the bots and the human player to the simulation defined by the chosen scenario.

The simulation will start, and you will see the market data updating in real-time in the watchlists and charts. You can interact with the simulation by placing orders manually, just as you would with a real broker.

Trading Bots

The JESSX simulator includes several types of trading bots, each with a different trading strategy. Bots are configured in the scenario XML and implemented in Java within the org.eclipsetrader.jessx plug‑in. Common bots include:

  • Discreet: A bot that trades based on market trends.
  • DiscreetIT: A variation of the Discreet bot.
  • NotDiscreet: A more aggressive trading bot.

The bots will automatically start trading once the simulation begins. You can observe their activity in the orders view.

Custom Scenarios

To add your own scenario:

  1. Create an XML file following the structure of the bundled scenarios.
  2. Place it under <workspace>/.metadata/.plugins/org.eclipsetrader.jessx/scenarios/ or use Browse… in Preferences to select it.
  3. Select it in Preferences → Plugins → JessX! Scenarios and apply the changes.
